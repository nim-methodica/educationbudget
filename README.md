# מערכת ניהול תקציב משרד החינוך

אפליקציית ווב מקומית בעברית לניהול הזמנות מסגרת, תקנות, הזמנות פרויקט, גביה חודשית ודוחות למרכבה.

## הרצה מקומית

```powershell
node server.js
```

לאחר ההרצה פותחים:

```text
http://localhost:4173
```

## מה כלול בגרסת MVP

- מסך ראשי נקי לפי תקנות.
- מעבר בין הזמנות מסגרת שנתיות וקביעת ברירת מחדל.
- הצגת סכומים תמיד ללא מע"מ וכולל מע"מ 18%.
- פירוט תקנה לפי הזמנות פרויקט רק לפי דרישה.
- הצגת פריטי הזמנה עם מספר סעיף, שם ועלות ללא מע"מ.
- שריון מלא של הזמנת פרויקט בעת הוספה.
- גביה חלקית ויתרה פתוחה לפי פריטים.
- התראות על יתרת פריט נמוכה מתחת ל-20%.
- תיק גביה חודשי לדוגמה עם חשבונית, הצמדה ותיעוד.
- הרשאות תצוגה בסיסיות: אדמין וצפייה בלבד.

## הערות להמשך

- חילוץ אוטומטי מלא מ-PDF/XLSX יתחבר לשכבת הייבוא הבאה.
- הדוחות החודשיים צריכים להמשיך להתבסס על התבניות הקשיחות הקיימות.
- בסיום הפיתוח המערכת מיועדת להעברה לשרת מאובטח, עם GitHub, סביבת פיתוח וסביבה חיה.
